start mongo local:
sudo systemctl start mongod
sudo systemctl status mongod
sudo systemctl restart mongod
sudo systemctl stop mongod
mongo

check current db: 
db

Listing databases/collections: 
show databases
show collections

Go to a particular database: 
use <your_db_name>

Creating a Collection:
Creating a Non-Capped Collection: db.createCollection("myCollection")
Creating a Capped Collection: db.createCollection("mySecondCollection", {capped : true, size : 2, max : 2})

Inserting Data:
db.myCollection.insertOne(
  {
    "name": "navindu", 
    "age": 22
  }
)

db.myCollection.insertMany([
  {
    "name": "navindu", 
    "age": 22
  },
  {
    "name": "kavindu", 
    "age": 20
  },

  {
    "name": "john doe", 
    "age": 25,
    "location": "colombo"
  }
])

Querying Data:
db.myCollection.find()
db.myCollection.find().pretty()

Updating documents:
db.myCollection.update({age : 20}, {$set: {age: 23}})

Removing a document:
db.myCollection.remove({name: "navindu"});

Removing a collection:
db.myCollection.remove({});
